Initial release of `@nhic/currantui-charts`, the CurrantUI data visualization package built on Apache ECharts. Ships the shared chart shell — show-table toggle, fullscreen, CSV/PNG/JPG export, HTML legend, and a source/attribution footer — plus the BarChart component (grouped/stacked, vertical/horizontal) and the token-driven theming core (`useEChart`, palette/theme readers, option builders).
